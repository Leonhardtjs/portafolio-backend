[MongoDB](https://www.mongodb.com)

# Requirements
Node v16

# Environment

Required variables to be used on your local development:

- PORT

# Setup

1. Clone the respository
```
git clone git@github.com:Leonhardtjs/portafolio-backend.git
```
2. Install dependencies

```
yarn install
```
3. Start application
```
yarn dev
```
